HTML Layouts

Create simple HTML layouts for your website.

<!DOCTYPE html>
<html lang="en">
<head>
    <title>{title}</title>
</head>
<body>
    {slot}
</body>
</html>

And re-use that layout in any number of pages.

<layout title="Title" src="main.html">

    <h1>Layouts Rock 🤘</h1>
    
</layout>
